The Apache Velocity DocBook Framework (DBF)

The DocBook Framework is intended to help you create high quality documentation suitable for online viewing and printing

Documentation for this framework is available in various formats:

  • PDF
  • HTML - single page (Download ZIP)
  • HTML - multiple pages (Download ZIP)
